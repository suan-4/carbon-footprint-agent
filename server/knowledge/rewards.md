# 碳迹同行商品权益库

## 1. 商品权益说明
- 商品权益分为实体商品和虚拟权益两类。
- 实体商品通常需要线下领取、统一发放或配送。
- 虚拟权益通常在兑换成功后进入已发放状态或待系统发放状态。

## 2. 商品清单

### 2.1 种子纸手账
- 商品名称：种子纸手账
- 类型：实体商品
- 所需积分：450
- 当前说明：再生纸封面，内含花草种子，书写完成后可继续种植。
- 领取方式：建议线下领取
- 适用场景：低碳记录、校园文创激励
- 订单状态示例：待领取

### 2.2 环保帆布袋
- 商品名称：环保帆布袋
- 类型：实体商品
- 所需积分：600
- 当前说明：适合校园通勤和购物，鼓励减少一次性塑料袋使用。
- 领取方式：建议线下领取或统一发放
- 适用场景：日常携带、环保宣传
- 订单状态示例：待领取

### 2.3 图书馆延时券
- 商品名称：图书馆延时券
- 类型：虚拟权益
- 所需积分：800
- 当前说明：适合考试周或项目冲刺阶段使用。
- 发放方式：兑换成功后由系统发放
- 适用场景：自习、备考、课程项目
- 订单状态示例：已发放

## 3. 商品字段建议
每个商品建议至少包含以下字段：
- 商品 ID
- 商品名称
- 分类
- 所需积分
- 当前库存
- 商品说明
- 发放方式
- 领取方式
- 是否支持兑换
- 订单状态说明

## 4. 兑换规则说明
- 用户积分余额需满足商品所需积分。
- 商品库存需大于 0。
- 兑换成功后系统立即扣减积分。
- 每次兑换会生成唯一订单号。

## 5. 订单状态建议
### 5.1 待领取
- 适用于实体商品
- 表示订单已创建，等待用户线下领取或等待后续发放安排

### 5.2 已发放
- 适用于虚拟权益
- 表示权益已发放到用户账户或系统已完成处理

### 5.3 已完成
- 可用于后续扩展
- 表示商品已领取、已核销或权益已实际使用完成

### 5.4 已取消
- 可用于后续扩展
- 表示订单被取消，是否退回积分需根据平台规则决定
