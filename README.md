# Carbon Trace - 个人碳排放足迹跟踪小程序

## 项目概述

本项目旨在开发一款适配个人场景的碳排放足迹跟踪小程序，解决日常碳排放记录、环保意识培养、低碳生活引导等问题，目标用户为关注环境保护、想要践行低碳生活的个人用户及环保倡导者。

## 功能介绍

### ① 碳排放记录

支持用户记录日常活动中的碳排放数据，包括出行方式（驾车、公交、地铁、骑行等）、饮食消费（肉类、乳制品、素食等）、能源使用（用电、燃气等），自动计算对应的碳排放量。

### ② 运动轨迹与数据分析

自动记录跑步、步行等运动的轨迹、运动时长和卡路里消耗，配合碳排放数据生成可视化图表，支持按日、周、月维度查看碳排放趋势和分布情况。

### ③ 环保目标设置

用户可设定个人碳减排目标，系统自动跟踪目标达成进度，提供阶段性成就反馈，帮助用户养成低碳生活习惯。

### ④ 排行榜与成就体系

按碳减排量进行排行榜展示，设置多种跑步/环保成就称号（如"低碳达人"、"健步如飞"等），发放荣誉标识激励用户持续参与。

### ⑤ 社交与分享

支持用户发布运动动态和环保心得，可将碳排放数据分享至好友或群聊，邀请朋友一起参与低碳生活挑战。

### ⑥ 环保建议与知识科普

基于用户的排放数据提供个性化减排建议，定期推送环保小知识和低碳生活技巧，帮助用户了解碳足迹相关知识。

- **开发语言**：TypeScript / JavaScript

## 快速开始

### 开发环境准备

1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 注册微信小程序账号，获取 AppID

### 本地运行

1. 克隆项目到本地
2. 使用微信开发者工具打开项目目录
3. 点击编译预览

### 构建发布

1. 在微信开发者工具中点击「上传」
2. 登录小程序管理后台提交审核
3. 审核通过后发布上线
